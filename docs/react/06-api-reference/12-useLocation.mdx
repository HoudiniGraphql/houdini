Returns the current location. Use this to read the pathname, route params, or query string, or to navigate imperatively.

```tsx
import { useLocation } from '$houdini'

export function ActiveLink({ href, label }: { href: string; label: string }) {
  const { pathname } = useLocation()
  return (
    <a href={href} aria-current={pathname === href ? 'page' : undefined}>
      {label}
    </a>
  )
}
```

## Reading the query string

`search` is the parsed query string of the current URL. Keys that match the page's nullable query variables are coerced to their scalar type (so an `Int` comes back as a number); any other keys are left as raw strings. A key that appears more than once becomes an array.

```tsx
import { useLocation } from '$houdini'

export function Pager() {
  // /search_params?offset=20&tab=reviews -> { offset: 20, tab: 'reviews' }
  const { search } = useLocation()
  return <span>Showing from {search.offset ?? 0}</span>
}
```

### Custom scalars

A declared search param backed by a [custom scalar](~/api/config#custom-scalars) is **unmarshaled** back to its runtime type — the inverse of how `<Link>` and `goto` marshal it into the URL. For example, with a `DateTime` scalar, `?after=1704067200000` comes back as a `Date`:

```tsx
const { search } = useLocation()
search.after // -> Date
```

> One consequence to know: the URL carries values as strings, so they're decoded with `JSON.parse` before unmarshaling. A custom scalar whose value is `"true"` or `"123"` will therefore arrive as a boolean or number. If a scalar must stay a string, give it a `marshal`/`unmarshal` that preserves that.

**Signature**

```ts
function useLocation(): {
  pathname: string
  params: Record<string, string>
  search: Record<string, any>
  goto: Goto
}
```

| Field | Type | Description |
|---|---|---|
| `pathname` | `string` | The current URL path (including the query string) |
| `params` | `Record<string, string>` | The current route params (untyped) |
| `search` | `Record<string, any>` | The parsed query string — declared search params coerced (and custom scalars unmarshaled), other keys raw, repeated keys as arrays |
| `goto` | `Goto` | Navigate to a URL imperatively — accepts a string or a typed `{ to, params, search }` target |
