---
title: Navigation
description: Linking between pages in Houdini's React framework
---

Use the `<Link>` component to navigate between pages. It renders a plain `<a>` element and accepts all standard anchor attributes alongside a typed `to` prop:

```tsx
import { Link } from '$houdini'

<Link to="/shows">All Shows</Link>
```

Houdini knows every valid route in your app, so `to` is type-checked at compile time. For parameterized routes, pass a `params` object:

```tsx
<Link to="/shows/[id]" params={{ id: show.id }}>
    {show.title}
</Link>
```

`<Link>` also supports [preloading](~/api-reference/link#preloading) and a [`disabled` prop](~/api-reference/link#disabled) for inert links. See the [`<Link>` API reference](~/api-reference/link) for the full list of props and examples.
